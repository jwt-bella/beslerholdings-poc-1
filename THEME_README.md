# Kelly Wisness Theme Variations

Five themed versions of Kelly's profile page using the Besler branding colors.

## Available Themes

| Theme | File | Colors |
|-------|------|--------|
| **Primary** (Default) | `kelly-wisness.html` | `#00AEEF` on dark backgrounds |
| **Orange** | `kelly-wisness-orange.html` | `#E25A2A` accent |
| **Teal** | `kelly-wisness-teal.html` | `#003448` with `#00AEEF` accents |
| **Light** | `kelly-wisness-light.html` | `#E6F7FD` background |
| **Dark** | `kelly-wisness-dark.html` | `#000000` with `#00AEEF` accents |

## Branding Colors Reference
- Primary: `#00AEEF` (Light Blue/Cyan)
- White: `#FFFFFF`
- Black: `#000000`
- Orange: `#E25A2A`
- Teal: `#003448`
- Light Blue: `#E6F7FD`
