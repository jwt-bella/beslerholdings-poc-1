# Besler Holdings Website Development - Day 1

## Project: beslerholdings-poc-1

### Major Progress
Built and deployed complete HTML landing page for Besler Holdings with the following features:

**Completed Sections:**
- Hero with gradient background and CTA
- About Us section
- Our Focus (Services) - 3 cards
- Portfolio - 3 cards (Tech, Healthcare, Real Estate)
- Latest News section
- Testimonials
- Leadership Team (7 members with photos)
- Contact Form
- Footer

**Theming:**
- Tailwind CSS via CDN
- Two themes: Light (Executive Navy) and Dark (Midnight Premium)
- Defaults to dark mode
- Sun/moon toggle in navbar
- CSS custom properties for theme variables

**Team Members (7 total):**
1. Phil Besler - Chairman of the Board
2. Wade Wright - Chief Executive Officer
3. Kristin DeGroat - Chief Legal Officer
4. Maria Palumbo - Chief Financial Officer
5. Jason Nadal - Information Security Officer
6. Eric Englebretson - VP of Information Technology
7. Kelly Wisness - VP of Marketing

**Bio Pages Created:**
- Individual bio pages for all 7 team members
- Photos uploaded and integrated for all members
- Content sourced from besler.com and LinkedIn
- Consistent styling with theme toggle

**Design Updates:**
- Mobile navbar shows full logo (emblem + text on desktop)
- Leadership cards show circular photos
- Hover effects with "View Profile →" links
- Eric's card centered in bottom row
- Footer version badge plain text (no highlight)

**Live Site:** https://jwt-bella.github.io/beslerholdings-poc-1/

### Technical Stack
- HTML5 with Tailwind CSS CDN

- GitHub Pages deployment
- No build step required
